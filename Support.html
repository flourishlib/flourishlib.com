Support

Looking for Support?

  • There is an IRC channel, #flourishlib on Freenode accessible via any IRC client or a simple web interface
  • The discussion forum allows your questions to be seen by lots of people
  • If youve found a bug or want to request a feature, you can create an issue in the appropriate repository on GitHub:
    • https://github.com/flourishlib/flourish-classes/issues
    • https://github.com/flourishlib/flourish-tests/issues
    • https://github.com/flourishlib/flourish-docs/issues

If you are interested in custom development, consulting or paid support/help, please feel free to email me.

Want to Support Flourish?

Below are some useful ways that you can help encourage the growth of Flourish, or just say thanks.

  • Help others in the discussion forum
  • Open issues or submit pull requests
  • Donate money via PayPal
  • Buy a gift from Amazon
